# gr-wwvb
WWVB in GNUradio

This project is a starting point for developing a WWVB flowgraph using GNUradio

The slides from the Cyberspectrum talk on April 29, 2015 are available here:
https://github.com/jasonabele/gr-wwvb/raw/master/WWVB_talk_Cyberspectrum_20150429.pdf

The original audio file used was found at the URL below and converted to WAV for ease of use with GNUradio
http://www.jks.com/wwvb/ant.wide.5min.15.625.kHz.44.1k.24b.au
